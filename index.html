Install Game

  • In a new browser tab open website Battle.net Account.
  • Create/Login a Battle.net account.
  • Remember the email, password, security question and security answer for future access to that Battle.net account.
  • In a new browser tab open website World of Warcraft: Subscription.
  • Select the 1 Month subscription and complete purchase.
  • Return to the Battle.net Account browser tab.
  • Select Games & Subscriptions and press Manage button near the World of Warcraft game.
  • On the Manage Subscription page press Cancel Subscription if you don't want to be billed automatically every month.
  • In the Battle.net Account browser tab press the Download Battle.net button and choose Download for Windows.
  • Run the downloaded Battle.net-Setup.exe to start installation of Battle.net desktop app.
  • During installation choose Launch Battle.net when you start your computer.
  • After installation Battle.net desktop app will start automatically.
  • Login your Battle.net account using email and password. Check the Keep me logged in button.
  • In the Favorites bar of the Battle.net desktop app click the World of Warcraft Classic product.
  • Above the Install button change GAME VERSION to World of Warcraft Classic (default is Cataclysm Classic).
  • Press the Install button to start installation of the game. Space required is around 5 GB.
  • Wait for game to complete installation.

  • Click Save and Exit button.

  • Click Continue as guest.

Game Settings

  • Start the game.
  • If you are warned about interface modifications which are out of date choose Load Anyway and confirm with Okay.
  • Change Realm to Firemaw PVP or any realm that has PVP type and Medium population. They are all connected.
  • Open Menu > System > Graphics settings.
    • Vertical Sync should be Enabled to not render more often than the monitor can display.
    • Set Graphics Quality to 1 then set:
      • Texture Resolution to High to see maximum texture detail.
      • Spell Density to Everything so you can see all spells and effects.
      • Environment Detail to 10 so you can see small objects farther.
    • In the Advanced section set:
      • Texture Filtering to 16x Anisotropic to reduce ground flickering.
      • Max Foreground FPS to 60 or the refresh rate of your monitor.
      • Max Background FPS to 60 or the refresh rate of your monitor.
      • Target FPS to 60 or the refresh rate of your monitor.
  • Open Menu > System > Audio settings.
    • Check Sound in Background.
    • Set Audio Channels to 128
