Battle.net desktop app

  • Open website Battle.net in browser.
  • Create a new (or login an already existing) Battle.net account using an email account that you can access.
  • Remember the email, password, security question and security answer for future access to that Battle.net account.
  • In the Battle.net account website press the Download Battle.net button and download the Windows version of the Battle.net desktop app.
  • Run the downloaded Battle.net-Setup.exe to start installation of Battle.net desktop app.
  • During installation choose Launch Battle.net when you start your computer.
  • After installation Battle.net desktop app will start automatically.
  • Login your Battle.net account using email and password. Check the Keep me logged in button.
  • In the Favorites bar of the Battle.net desktop app select the World of Warcraft Classic product.
  • Above the Install button select the World of Warcraft Classic game version (default is Cataclysm Classic).
  • Press the Install button to start installation of the game. Space required is around 5 GB.
