Battle.net desktop app

  • Open website Battle.net in browser.
  • Create a new (or login an already existing) Battle.net account using an email account that you can access. Remember the email, password, security question and security answer for future access to that Battle.net account.
  • In the Battle.net account website press the Download Battle.net button and download the Windows version of the Battle.net desktop app.
  • Run the downloaded Battle.net-Setup.exe to install Battle.net desktop app

Addons
